# Computer Systems

本 README 文件为 `computer-systems` 目录提供说明。

## 简介

此目录包含了与计算机系统核心概念相关的代码示例和学习材料。内容覆盖了并发、网络编程、虚拟内存等多个主题，旨在为学习“深入理解计算机系统”（Computer Systems: A Programmer's Perspective）等课程提供实践支持。

## 模块介绍

本代码库按主题划分为以下几个模块：

*   **`concurrent/`**: 包含与并发编程相关的示例，例如多线程、同步机制（锁、信号量）等。
*   **`exceptional-control-flow/`**: 涉及异常控制流的例子，包括中断、信号和异常处理。
*   **`linking/`**: 包含与链接过程相关的示例，解释了静态链接和动态链接的原理。
*   **`network/`**: 包含网络编程的示例，例如套接字（Socket）编程和基本的客户端/服务器模型。
*   **`system-level-io/`**: 提供了系统级 I/O 操作的例子，展示了文件描述符和 I/O 重定向等概念。
*   **`virtual-memory/`**: 包含与虚拟内存管理相关的代码，例如内存分配和地址翻译。
*   **`libs/`**: 存放了项目共享的库文件和头文件。

## 如何使用

每个子目录都包含独立的示例代码，通常为 C 语言。要编译和运行这些示例，您需要一个 C 编译器（如 `gcc`）。

### 编译示例

进入任意一个子目录，使用 `gcc` 编译源代码。例如，在 `network/` 目录中：

```bash
cd network/
gcc -o client client.c
gcc -o server server.c
```

建议为每个子目录创建独立的 `Makefile` 以简化编译过程。

### 运行示例

编译后，您可以直接运行生成的可执行文件。

```bash
./server 8080
./client 127.0.0.1 8080
```

具体的编译和运行命令请参考每个子目录内可能存在的说明。

## 开发

您可以自由地修改和扩展这些示例，以加深对计算机系统概念的理解。例如：

*   在 `concurrent/` 中尝试不同的同步策略。
*   在 `network/` 中实现一个简单的应用层协议。
*   在 `virtual-memory/` 中编写一个自己的内存分配器。
